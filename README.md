# Análisis de K-means en Datos de FBREF - SCA en la Premier League

## Descripción
Este notebook implementa un análisis de clustering utilizando el algoritmo K-means en datos de "Shot-Creating Actions" (SCA) de la Premier League obtenidos de FBREF. El objetivo es segmentar jugadores en distintos grupos según sus acciones de creación de oportunidades de gol.

## Uso
1. Descarga el dataset de FBREF con información sobre SCA en la Premier League.
2. Ejecuta el notebook paso a paso para:
   - Cargar y preprocesar los datos.
   - Normalizar las variables relevantes.
   - Aplicar el algoritmo K-means para segmentar jugadores.
   - Visualizar los clusters generados.
3. Interpreta los resultados obtenidos para identificar patrones en la creación de jugadas ofensivas.

## Resultados Esperados
El análisis permitirá agrupar jugadores en clusters según su contribución a la generación de jugadas de gol. Esto podría ayudar a comparar perfiles de jugadores, identificar patrones en su desempeño y generar insights sobre su rol en el equipo.
